# Week 3 Capstone Reflection

### Why do remote repositories matter?
Remote repositories are essential because they prevent data loss by keeping code securely backed up in the cloud. They allow mentors to inspect your work, provide feedback, and enable multiple developers to collaborate smoothly on a shared project layout.

### What does the -u flag do?
The -u (upstream) flag links your current local branch to the remote branch on GitHub. Once you run it the first time, Git remembers where to send your files, allowing you to simply type git push or git pull in the future without specifying the remote name and branch every single time.

### How do local and remote repositories stay synchronized?
They stay synchronized by running git push to upload local work to the remote hub, and running git pull to bring any updates made on the remote side down into your local tracking branch environment.
