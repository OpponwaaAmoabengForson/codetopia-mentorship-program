##  Core Git Concepts
* Git Clone: Copies an entire remote repository from GitHub down to your machine for the very first time, creating a brand new local directory with all history and branches.
* Git Pull: Fetches and downloads updates from an existing remote repository into your current active local branch, keeping your files in sync without starting over.
